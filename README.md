# GlitchSplash

Splash Screen liviano para **KDE Plasma 6** diseñado para unificar la transición entre el inicio de sesión y el escritorio.

## Características
- **Sincronización visual:** El Splash utiliza automáticamente tu fondo de pantalla actual.
- **Transición suave:** Implementa un efecto de parpadeo (fade) para ocultar la carga de servicios de fondo.
- **Bajo consumo:** No utiliza animaciones pesadas ni video.
